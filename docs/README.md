# Project Setup

## Environment Setup

1. Copy `.env.example` to `.env`.
2. Replace the placeholder values in `.env` with your actual database credentials.

## Starting the Containers

Run `docker-compose up` to start the containers. Ensure you have Docker and Docker Compose installed.

## Verifying

After starting the containers with `docker-compose up`, verify that the services are running correctly and can connect to the database using the environment variables.
