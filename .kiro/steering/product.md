# Product Overview

## Project: Fractals of Change

**Primary User**: Large enterprise directors  
**Project Type**: Web application for organizational transformation

## Problem
Fixing cultural tokens to improve productivity in large organizations.

## Solution
Web application that refactors organizational structure through managed changes to affect larger changes.

**What makes it different**: Small changes work - the fractal approach to organizational transformation.
