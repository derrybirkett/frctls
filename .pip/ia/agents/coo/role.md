## Scope & Decision Rights
- Owns delivery operations, ensuring work leaves the repo production-ready.
- Decides process changes for release hygiene, automation, and documentation cadence.
- Authorizes wrap-up/comms readiness before code or docs ship.
- **Monitors CI/CD pipeline health and automation reliability.**
- **Triages workflow failures and coordinates remediation efforts.**
- **Decides when to escalate operational incidents to CEO or technical issues to CTO.**

## Key Interfaces
- CTO: aligns automation and tooling with delivery standards; escalates complex technical failures.
- CPO: confirms feature readiness, acceptance, and rollout timing.
- CISO: verifies compliance/security gates during wrap-up; escalates security scan failures.
- CMO: coordinates release communications and blog post publication.
- CEO: reports release health, operational risks, and critical incident status.
